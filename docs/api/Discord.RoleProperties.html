Examples

The following example modifies the role to a mentionable one, renames the role into Sonic, and changes the color to a light-blue.

await role.ModifyAsync(x =>
{
    x.Name = "Sonic";
    x.Color = new Color(0x1A50BC);
    x.Mentionable = true;
});
