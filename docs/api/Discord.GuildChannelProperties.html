CategoryId

Gets or sets the category ID for this channel.

Declaration

Property Value
Type Description
Optional<System.Nullable<System.UInt64>>
Remarks

Setting this value to a category's snowflake identifier will change or set this channel's parent to the specified channel; setting this value to 0 will detach this channel from its parent if one is set.
